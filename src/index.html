Turkmen Linux is a standalone based GNU/Linux distribution. It uses its own YMP (Yerli ve Milli Package) manager. All packages come with header files and static libraries. In this way, it is easier to compile than other distributions.

It has a mixed (binary and source) package system that works quickly. While those who want to install packages from the source code, those who want can use the compiled packages.
